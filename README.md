# herrmedia Next.js

A minimal Next.js portfolio site with Tailwind CSS, based on the original herrmedia.com design.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3300](http://localhost:3300) in your browser.

## Project Structure

- `app/` - Next.js app directory
  - `layout.tsx` - Root layout with fonts
  - `page.tsx` - Main page component
  - `globals.css` - Global styles and Tailwind directives
- `public/` - Static assets
  - `fonts/` - DIN Condensed Bold font files
  - `img/` - Images and videos

## Build

```bash
pnpm build
pnpm start
```
